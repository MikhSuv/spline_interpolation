# Интерполяция функций кубическими полиномами

В корневой директории команда `make` создаст исполнимые файлы: `spline_interpolation` и `spline_test`.

## Основная программа `spline_interpolation`

Использование:
```bash
./polyinterp [q]
```
Необязательный аргумент `q` - целое число, задающее значение `q`(По умолчанию `q = 100`).

Для построения графика написан gnuplot скрипт `plot.plt`.
Он строит график сплайна и наносит на него исходные точки.
Для построения графика:
```bash
gnuplot plot.plt
```
или
```bash
make plot
```

## Программа `spline_test`
Выполняет интерполяцию функции со случайными шумами.
Записывает в `original.dat` исходные точки функции,
а в `spline.dat` - узлы и значения полученные интерполяцией.
По этим данным строится график `spline.pdf`.
